## 📚 CHAPTER 9: Troubleshooting

### "Command not found"
**Problem:** Your computer doesn't know where xmt-cli is.

**Solution:** Make sure you're in the right folder:
```bash
cd /home/pepo/Desktop/xmt-cli
```

Then try again with `./target/release/xmt-cli`

### "Permission denied"
**Problem:** The computer won't let you run it.

**Solution:** Ask a grown-up to run:
```bash
chmod +x ./target/release/xmt-cli
```

### "Connection refused"
**Problem:** The blockchain network isn't responding.

**Solution:** Use `--local-only` to work offline:
```bash
xmt-cli ritual --apex 936 --local-only --intent "My wish"
```

### Still Stuck?
Ask for help! The Lattice community is always ready to assist young sovereigns.
